# object-pick-ts [![NPM version](https://img.shields.io/npm/v/object.pick.svg?style=flat)]


## Install
Install with npm:

```$ npm install --save object-pick-ts```

## Usage

```ts
var pick = require('object-pick-ts');
 
let result = pick({a: 'a', b: 'b', c: 2}, ['a', 'c']);

//{a: 'a', c: 2};
```
